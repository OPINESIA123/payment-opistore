Informasi Bot Telegram

Untuk mengintegrasikan dengan bot Telegram, Anda perlu mengisi token bot dan chat ID tujuan di bawah ini.

Cara mendapatkan Token Bot:

  1. Buka Telegram dan cari @BotFather
  2. Ketik /newbot untuk membuat bot baru
  3. Ikuti instruksi sampai selesai
  4. Salin token yang diberikan

Cara mendapatkan Chat ID:

  1. Tambahkan bot Anda ke grup/channel
  2. Kirim pesan ke bot/channel
  3. Buka: https://api.telegram.org/bot<TOKEN_BOT_ANDA>/getUpdates
  4. Cari "chat":{"id":<CHAT_ID>} di hasil JSON
